# A simple image listing node.js app

## How to run the app?

### 0. Install dependencies

```bash
$ npm i
```

### 1. Configure environment variables

Copy the example dotenv file and modify it's values to suit your needs:
```bash
$ cp .env.example .env
$ vim .env # ... or whatever editor you use
```

### 2. Run the tests

```bash
$ npm test
```

### 3. Run the app!

```bash
$ npm start
```
